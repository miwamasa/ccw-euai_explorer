# 欧州AI法条目エクスプローラー・エディタ

欧州AI法の条文を閲覧・編集・管理するためのWebアプリケーションです。

## 機能

### 主要機能
- ①**条目コンテンツ(JSON)の読み込み・書き出し**: JSONファイルからデータを読み込み、編集後にエクスポート可能
- ②**条目インデックス表示**: 左ペインに全条目のインデックスをリスト表示
- ③**条目内容の表示・編集**: 右ペインで条目の詳細を構造に従って表示・編集
- ④**条目の追加・削除**: 新規条目の差し込みと既存条目の削除が可能
- ⑤**スライド生成**: 条目内容からプレゼンテーション用スライドを自動生成

### その他の機能
- **検索機能**: キーワードで条目を検索
- **フィルタリング**: カテゴリとリスクレベルでフィルタリング
- **関連条文ナビゲーション**: 関連条文へのワンクリック遷移
- **メタデータ管理**: 作成日時、更新日時、バージョン等の管理

## 使用方法

### 起動方法

1. **ローカルサーバーで起動**:
   ```bash
   # Pythonを使う場合
   python -m http.server 8000
   # または Python 2の場合
   python -m SimpleHTTPServer 8000

   # Node.jsのhttp-serverを使う場合
   npx http-server -p 8000
   ```

2. **ブラウザでアクセス**:
   ```
   http://localhost:8000
   ```

### 基本操作

#### データの読み込み
- アプリケーション起動時に自動的に `data/eu_ai_act_articles_complete.json` を読み込みます
- 「JSONデータ読み込み」ボタンで別のJSONファイルを読み込むこともできます

#### 条目の閲覧
1. 左ペインのリストから条目を選択
2. 右ペインに条目の詳細が表示されます
   - 条文本文（日英）
   - 要件リスト
   - 関連条文
   - 関連前文
   - 関連付属書
   - メタデータ

#### 条目の編集
1. 条目を選択後、「編集モード」ボタンをクリック
2. 条文本文を編集
3. 「表示モード」ボタンで編集を確定

#### 条目の追加
1. 「新規条目追加」ボタンをクリック
2. 条目番号を入力（例: 24条）
3. 新規条目が追加され、編集可能になります

#### 条目の削除
1. 削除したい条目を選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「OK」を選択

#### データの保存
- 「JSON保存」ボタンで編集内容をJSONファイルとしてダウンロード

#### スライド生成
1. 「スライド生成」ボタンをクリック
2. 全条目からスライドが自動生成されます
3. モーダルウィンドウでスライドをプレビュー
4. 「HTMLダウンロード」ボタンでスライドをHTMLファイルとして保存

### 検索・フィルタリング
- **検索ボックス**: 条目番号、タイトル（日英）で検索
- **カテゴリフィルタ**: 定義、範囲、禁止事項、提供者の義務等で絞り込み
- **リスクレベルフィルタ**: 禁止、高リスク、汎用AI等で絞り込み

## データ構造

### JSONファイル形式
```json
{
  "schema_version": "1.0",
  "description": "欧州AI法 P23～P70の全条文データ",
  "created_at": "2025-11-04T13:59:51.597357",
  "articles": [...]
}
```

### 条文オブジェクト
各条文は以下の情報を含みます:
- 基本情報（ID、条文番号、タイトル等）
- 条文本文（日英）
- 要件リスト
- 関連条文・前文・付属書
- メタデータ（作成日時、ステータス等）

詳細は `doc/article_schema_proposal.md` を参照してください。
